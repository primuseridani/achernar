agbsum

agbsum is a command line utility for patching AGB images.

compatibility

agbsum is written in C99 and uses makefiles as its build system. It has been tested to compile under Clang, GCC, and the Tiny C Compiler (altough the latter may have problems with the standard library), as well as both GNU Make and BSD Make.


In theory, all UNIX-like systems (including older ones) should be supported. Please open a bug report if you experience otherwise.
